# 项目复利与踩坑日志

## 2026-07-08

- 项目定位：Project 004 选择 `QuoteCash`，一个自由职业者报价单 / 发票 / 收款文案生成器，优先做成无需后端的静态工具。
- 实现取舍：第一版只做本地编辑、计算、打印、导入导出与收款链接承载，不接真实支付 API；真实收款通过用户自己的 Stripe Payment Links、Gumroad、微信/支付宝链接解决。
- 注意事项：`localStorage` 在直接用 `file://` 打开时不同浏览器行为可能不完全一致；正式发布建议使用 GitHub Pages 或其他 HTTP/HTTPS 静态托管。
